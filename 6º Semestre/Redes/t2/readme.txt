-Para compilar abra o terminal na passta dos arquivo, então use o comando: ./compile.sh

-Configuracoes importantes (Constantes em "funcoes.h")

-Nas tabelas de roteamento, o numero entre [colchetes] significa o custo total até o destino, e o dentro do
  (parenteses) o próximo salto.

-Antes de alterar a topologia da rede, altere a quantidade de nós (NROUT) no arquivo "funcoes.h"

-A topologia da rede pode ser alterada no arquivo enlaces.config

-Informacoes sobre a porta utilizada por cada roteador, seus endereços, estão no arquivo
roteador.config

-Cada roteador deve ser instanciado executando ./roteador ID em um terminal diferente,
onde ID é o numero do roteador instanciado

-Uma imagem contendo a topologia default da rede acompanha este arquivo
